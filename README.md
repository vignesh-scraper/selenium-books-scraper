# Selenium Books Scraper

This project is an automated web scraping script built using **Python** and **Selenium**.

It navigates through all pages of the Books to Scrape website and extracts:

- Book Title
- Price
- Rating

The script collects data from **1000 books** and can be extended to save results into CSV or Excel for analysis.

---

##  Features

- Automated browser control using Selenium  
- Pagination handling across all website pages  
- Structured data extraction  
- Beginner-friendly clean Python code  
- Ready for real-world scraping projects  

---

##  Technologies Used

- Python

- WebDriver Manager
